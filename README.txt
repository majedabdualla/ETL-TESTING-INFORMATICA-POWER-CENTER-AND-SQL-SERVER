# ETL-TESTING-INFORMATICA-POWER-CENTER-AND-SQL-SERVER
This repository demonstrates an ETL (Extract, Transform, Load) process using Informatica PowerCenter with data sourced from SQL Server. The project involves:
Source: Three tables in SQL Server
ETL Tool: Informatica PowerCenter for data extraction, transformation, and loading
Target: A destination database where transformed data is loaded

 
# Key Features:

Connect to SQL Server source tables

Perform transformations (e.g., data cleaning, joins, calculations) in Informatica

Load transformed data into a target database

Showcase end-to-end ETL workflow
